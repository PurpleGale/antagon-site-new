# Описание
Хранит уровни опыта, при использовании обменивает опыт местами. При использовании с ножницами (только ур. 42+)ломается, становясь [[Дефектный кристалл памяти.]].

| Начальное значение | 42  |
| ------------------ | --- |

# Используется для
[[Дефектный кристалл памяти.]]
# Получение
Зарядка опытом кристалла памяти.
