# Описание
Спавнит ловушку из ветра. Когда игрок касается, его побрасывает наверх блоков на 10. По-возможности с сохранением инерции. После использования ловушка выпадает как дроп. Можно добавить вариацию **Вечный ветер**, которую как и первую, но и только, можно демонтировать любым ударом.
