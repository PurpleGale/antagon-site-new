# Описание
Эквивалент незеритовой, при полном сете даёт иммунитет к замедлению и скорости, также полностью(или почти) нивелирует отдачу от ударов.

| Перезарядка     | 2 с                                 |
| --------------- | ----------------------------------- |
| Активация       | Периодическая (полный сет)          |
| Броня           | 3, 8, 6, 3 (незерит)                |
| Снятие эффектов | отдача(удары), замедление, скорость |
| Зачаровываемое  | Да                                  |
