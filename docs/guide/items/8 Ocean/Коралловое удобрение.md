# Описание
*Мочалка*
Расходник. Пытается создать(строго в воде) коралл с соответствующей стороны кораллового блока. При нажатии на коралл превратит его в цельный блок. Работает только с живыми кораллами. Артефакт на базе лаймового красителя, пкм не отменять. Крафт: ламинария + костная мука
