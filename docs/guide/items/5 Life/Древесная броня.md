#Одежда #Не_сделано
*Из берёзовой коры*
# Описание
В жарких биомах: +1 броня
В умеренных: периодически немного лечит
В холодных: иммунитет к замедлению
Везде: удвоенный урон от огня

| Активация                     | При полном сете                     |
| ----------------------------- | ----------------------------------- |
| Прочность                     | не указано                          |
| Броня                         | 2, 5, 4, 1(кольчужная, сверху вниз) |
| Разрешена наковальня          | Нет                                 |
| Зачаровываемое                | Да                                  |
| Множитель огненного урона     | 2                                   |
| Снятие эффектов               | замедление                          |
| Периодичность снятия эффектов | 1с                                  |
| Лечение                       | 1                                   |
| Периодичность лечения         | 15с                                 |
| Увеличение брони              | 1(каждый элемент)                   |
