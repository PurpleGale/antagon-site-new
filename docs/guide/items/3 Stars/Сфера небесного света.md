# Описание
Ставится на землю, накладывает свечение на всех поблизости и снимает слепоту. Работает под открытым небом.

| Активания                | Пока поставлено    |
| ------------------------ | ------------------ |
| Частота эффекта          | 1с                 |
| Радиус наложения эффекта | 20 блоков          |
| Радиус снятия эффекта    | 10 блоков          |
| Наложение эффекта        | свечение           |
| Длительность эффекта     | 10с                |
| Снятие эффектов          | тьма, слепота      |
| Ограничение работы       | под открытым небом |
# Используется для
[[Лунный амулет]]
