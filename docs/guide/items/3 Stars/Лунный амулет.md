| ----------- | ------------- |
| Новолуние   | Невидимость   |
| Растущая 1  | Регенерация   |
| Растущая 2  | Регенерация   |
| Растущая 3  | Регенерация   |
| Полнолуние  | Сопротивление |
| Убывающая 3 | Спешка        |
| Убывающая 2 | Спешка        |
| Убывающая 1 | Спешка        |
