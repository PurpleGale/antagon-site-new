# Описание
### Огненный меч
При ударе иссушает цель, может поджечь.

| Активация          | удар               |
| ------------------ | ------------------ |
| Наложение эффектов | цель, всегда       |
| Длительность       | не указано         |
| Эффект             | иссушение, горение |
### Водный меч
При ударе ослабляет цель цель.

| Активация          | удар         |
| ------------------ | ------------ |
| Наложение эффектов | цель, всегда |
| Длительность       | не указано   |
| Эффект             | слабость     |
### Ядовитый меч
При ударе отравляет цель.

| Активация          | удар         |
| ------------------ | ------------ |
| Наложение эффектов | цель, всегда |
| Длительность       | не указано   |
| Эффект             | отравление   |
### Гравитационный меч
При ударе искажает гравитацию цели.

| Активация          | удар         |
| ------------------ | ------------ |
| Наложение эффектов | цель, всегда |
| Длительность       | не указано   |
| Эффект             | не указано   |
