# Описание
Подозрительное яйцо, Ядовитое яйцо, Гравитационное яйцо, Водное яйцо, Огненное яйцо. Ведут себя как мяч для водного поло. Первое падает с Звёздного Дракона (но кривовато если честно), остальные создаются из первого если скормить ему 64 блока слизи/аметиста/призмарина/светокамня. Особые яйца имеют небольшую ауру вокруг них, схожую с пилонами, но слабее (отравление, отсутствие гравитации, постоянный неблокируемый слабый урон, поджог). В мифике искать по <тип>DragonEggItem (<тип> = Empty,Poison,Gravity,Water)
