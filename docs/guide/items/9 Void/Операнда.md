# Описание
*Арифмометр плоти*
Прокручивает вариации на пкм, невозможно при перезарядке, кд 1с, использование по взмаху, изначально стоит на умножении.
Плюс: +1 хп, кд 1с
Минус: -1 хп, кд 1с
Слеш: уполовинивает хп, кд 20 секунд
Крест: удваивает хп, кд 20 секунд
# Используется для
[[Конь в кубе]]
