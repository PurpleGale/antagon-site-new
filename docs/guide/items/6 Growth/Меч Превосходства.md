#Оружие #Не_сделано

# Описание
урон и прочность небольшие, крафт дорогой, но при ударе увеличивает урон на +1 (сбрасывается при 3 секундах без ударов). После 10 ударов, каждый удар бьёт ещё и владельца.

| Активация             | При ударе       |
| --------------------- | --------------- |
| Урон                  | 4 (дерево)      |
| Прочность             | 60 (дерево)     |
| Перезарядка           | нет             |
| Длительность          | 3с              |
| Увеличение урона      | 1               |
| Перегрузка            | более 10 ударов |
| Самоурон в перегрузке | Да              |
| Зачаровываемое        | Да              |
